Listas Ordenadas e não ordenadas

listas Ordenadas

listas ordenadas sao listas com a tag OL e existes 5 tipos
        
    
            
  1.  type - 1.         lista numerica
            
  2.  type - A.         Lista alfabetica
            
  3.  type - a.         Lista alfabetica minúscula 
            
  4.  type - I.         Lista algarismo romanos
            
  5.  type - i.         Lista algarismo romanos minuscula

Listas nao ordenadas

listas nao ordenadas sao listas com a tag UL e existem tres tipos de listas 

        
    
            
  •  type - disc.         Formato de disco •
            
  •  type - circle.       Formato de circplo °
            
  •  type - square.       Formato de quadrado ▪
